## Descrição do Projeto

Este é um aplicativo de rede social/comunidade completo e profissional desenvolvido em React Native. Ele funciona como uma plataforma de usuários. O projeto foi desenvolvido como trabalho acadêmico, demonstrando conhecimentos em desenvolvimento mobile, interface responsiva e gerenciamento de estado em React Native.

Funcionalidades Principais:
1. Autenticação 🔐
Criar conta com email e senha
Fazer login seguro
Logout com confirmação
Sessão persistente durante o uso
2. Dashboard (Tela Principal) 📰
Lista de 10 posts da comunidade
Cada post mostra: título, resumo e ID
Pull-to-refresh para atualizar
Indicador de carregamento
Tratamento de erros
3. Detalhes do Post 📖
Visualizar post completo
Ver conteúdo integral
Informações do autor
Metadados (ID do post e do usuário)
Botão de voltar funcional
4. Perfil do Usuário 👤
Avatar com inicial do email
Email exibido
Estatísticas (posts, seguidores, seguindo)
Menu de configurações
Botão de logout
5. Configurações ⚙️
Editar Perfil: Alterar nome e bio
Notificações: Ativar/desativar push e email
Privacidade: Perfil privado e status online
Modo Escuro/Claro: Toggle na tela de perfil
Tecnologias Usadas:
React Native com Hooks (useState, useEffect)
Custom Hooks (useAuth, useFetchPosts)
Tab Navigation (Início e Perfil)
API JSONPlaceholder para posts
Firebase Authentication (opcional)
Tema adaptativo (Claro/Escuro)
